# 1. 说明

```bash
# 创建帐号
geth account new

# 列出帐号
geth account list --keystore ~/.ethereum/keystore

# 私钥创建 帐号
geth account import <keyfile>

# 肯定是需要密码的
geth account import --password <passwordfile> <keyfile>

# 查询余额
web3.fromWei(eth.getBalance(eth.coinbase), "ether")
```
