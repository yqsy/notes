# 2. 常用

```bash
# 包含ISO-8859或ASCII
find . -name '*.h' -type f -print0 | xargs -0 file | grep -E 'ISO-8859|ASCII'

# 不包含ISO-8859和ASCII
find . -name '*.h' -type f -print0 | xargs -0 file | grep -E -v 'ISO-8859|ASCII'
```
