# 2. 实践     
```bash
yum install httpd -y

sudo mkdir /var/log/frp
sudo chown $(id -u):$(id -g) /var/log/frp
nohup ./frps -c ./frps.ini | rotatelogs /var/log/frp/frp.log-%Y%m%d 86400 &

```
